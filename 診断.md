# アプリが動かない場合の診断方法

## 🔍 ステップ1: ブラウザのコンソールを開く

1. ブラウザで `index.html` を開く
2. **F12キー**を押す（Macの場合は **Cmd+Option+I**）
3. **Console（コンソール）タブ**をクリック

## 📋 確認すること

### ケース1: 何もメッセージが表示されない

**原因:** JavaScriptファイルが読み込まれていない

**解決策:**
- ブラウザで**Ctrl+Shift+R**（強制リロード）を試す
- ファイルパスが正しいか確認
- `app.js`、`style.css`、`style_users.css` が同じフォルダにあるか確認

### ケース2: 🚀 App starting... が表示されるが途中で止まる

**表示されるメッセージを確認:**

- `📦 Opening database...` → データベースを開いています
- `✅ Database opened` → データベースOK
- `🎨 Updating UI...` → UIを更新中
- `📋 Rendering list...` → リストを描画中
- `✅ App initialized successfully` → 初期化完了！

**どこで止まっているかで原因が分かります**

### ケース3: ❌ エラーメッセージが表示される

**エラーメッセージをコピーして確認してください**

よくあるエラー:
- `Cannot read property ... of null` → HTML要素が見つからない
- `Failed to execute 'transaction' on 'IDBDatabase'` → データベースエラー
- `Uncaught ReferenceError` → 関数が定義されていない

## 🧪 簡易テスト

`test.html` を開いてみてください:
- ボタンが押せる → HTML/JavaScriptは動作している
- ボタンが押せない → 根本的な問題がある

## 🔧 よくある問題と解決策

### 問題1: ページが真っ白

**原因:** CSSファイルが読み込まれていない

**解決策:**
```
ブラウザのネットワークタブで確認:
- style.css (赤い×がついていないか)
- style_users.css (赤い×がついていないか)
```

### 問題2: +ボタンが反応しない

**原因:** イベントリスナーが登録されていない

**コンソールで確認:**
```javascript
document.getElementById('fab-add')
```

これで `null` が返ってくる場合、HTML要素が存在しません。

### 問題3: 以前のGoogle Drive版のキャッシュが残っている

**解決策:**
1. ブラウザのデベロッパーツールを開く (F12)
2. **Application** タブ (または **Storage** タブ)
3. **Local Storage** → サイトを右クリック → **Clear**
4. **IndexedDB** → **DanshariDB** を右クリック → **Delete database**
5. ページをリロード (**Ctrl+Shift+R**)

## 📸 スクリーンショット

もしよければ、以下のスクリーンショットを送ってください:
1. ブラウザのコンソールタブ
2. エラーメッセージ（もしあれば）

## 🆘 緊急用: 最小限のテスト

以下のHTMLを新しいファイル `simple-test.html` として保存して開いてみてください:

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Simple Test</title>
    <style>
        body { font-family: sans-serif; padding: 20px; }
        button { padding: 10px 20px; font-size: 16px; }
    </style>
</head>
<body>
    <h1>簡易テスト</h1>
    <button onclick="alert('動作OK!')">クリックテスト</button>

    <script>
        console.log('JavaScript is working!');

        // IndexedDB test
        const request = indexedDB.open('TestDB', 1);
        request.onsuccess = () => {
            console.log('IndexedDB is working!');
            alert('ブラウザは正常に動作しています');
        };
        request.onerror = () => {
            console.error('IndexedDB failed');
            alert('IndexedDBが使えません');
        };
    </script>
</body>
</html>
```

このテストで動作すれば、ブラウザ自体は問題ありません。
