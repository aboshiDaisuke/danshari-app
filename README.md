# 📱 断捨離アプリ - スマホ用ガイド

シンプルで無料の断捨離記録アプリです。手放したものを写真と一緒に記録できます。

## ✨ 特徴

- 📸 **写真撮影** - スマホのカメラで直接撮影できます
- 💾 **自動保存** - ブラウザに自動的に保存されます（見えない場所に安全に保存）
- 👥 **複数ユーザー** - 家族で共有できます
- 📦 **簡単バックアップ** - ZIP形式で写真をダウンロード可能
- 🌐 **オフライン対応** - インターネットなしでも使えます
- 🆓 **完全無料** - 複雑な設定不要、課金なし

## 🚀 使い方

### 1. ホーム画面に追加（推奨）

アプリをホーム画面に追加すると、ネイティブアプリのように使えます。

**iPhone/iPad (Safari):**
1. Safari でアプリを開く
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

**Android (Chrome):**
1. Chrome でアプリを開く
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

### 2. アイテムを追加

1. 右下の **+** ボタンをタップ
2. カメラアイコンをタップして写真を撮影
3. 手放す理由を選択
4. コメントを入力（任意）
5. 「手放す」ボタンをタップ

### 3. 記録を見る

- タップして詳細を表示
- 編集や削除も可能

## 💾 データのバックアップ

### 方法1: ZIP形式でダウンロード

1. 設定（⚙️）を開く
2. 「📤 写真のバックアップ」セクション
3. 「📥 写真をダウンロード」をタップ
4. ZIPファイルがダウンロードされます

### 方法2: Googleドライブに手動アップロード

1. ZIPファイルをダウンロード
2. Googleドライブアプリを開く
3. 「+」→「アップロード」
4. ダウンロードしたZIPファイルを選択

これで、自分のGoogleドライブに安全にバックアップできます。

### 方法3: PC内のフォルダに自動保存（PC/Mac）

1. 設定（⚙️）を開く
2. 「保存先フォルダの設定」セクション
3. 「📂 保存先フォルダを選択」をクリック
4. フォルダを選択すると、以降自動で写真が保存されます

※Chrome/Edge推奨（Safariは非対応）

## 👥 ユーザー管理

### ユーザーの切り替え

1. 画面上部の現在のユーザー名をタップ
2. 別のユーザーを選択

### 新しいユーザーの追加

1. ユーザー選択画面を開く
2. 下部に新しいユーザー名を入力
3. 「追加」をタップ

### ユーザーの削除

1. ユーザー選択画面を開く
2. 右上の「編集」をタップ
3. 削除したいユーザーの「削除」ボタンをタップ

## 🔧 設定

### データのリセット

現在のユーザーの記録をすべて削除できます。

⚠️ **注意**: 削除したデータは復元できません。必要に応じて事前にバックアップしてください。

## 📊 データの保存場所

### ブラウザ内（IndexedDB）- 見えない安全な場所

- 写真とメタデータはブラウザ内の専用領域に保存されます
- 普通のファイルとしては見えませんが、アプリ内で安全に管理されています
- オフラインでも動作します
- ブラウザのデータを削除すると消えます（定期的にバックアップを推奨）

### PC内のフォルダ（オプション）

- 設定で指定したフォルダに写真が保存されます
- ブラウザを閉じても残ります
- 通常のフォルダとして管理できます

## 💡 Tips

### スマホで快適に使う

- ホーム画面に追加すると、アドレスバーが非表示になります
- オフラインでも使えます
- 定期的にバックアップをとりましょう

### バックアップのタイミング

- 月に1回程度
- 大量にアイテムを追加した後
- 機種変更やブラウザ変更の前

### ストレージ容量

- 1枚の写真は約100KB〜500KB
- 100枚で10MB〜50MB程度
- ブラウザのストレージ制限に注意

## ❓ よくある質問

### Q: データはどこに保存されますか？

A: ブラウザ内（IndexedDB）に保存されます。オプションでPC内のフォルダにも保存できます。

### Q: オフラインで使えますか？

A: はい、インターネット接続なしでも使えます。

### Q: 他のデバイスと同期できますか？

A: 直接の同期機能はありませんが、ZIPでバックアップして別デバイスにインポートできます。

### Q: ブラウザを変えたらデータは消えますか？

A: はい、データはブラウザごとに保存されます。事前にバックアップをとってください。

### Q: 機種変更したらどうなりますか？

A: 新しい端末ではデータが引き継がれません。事前にZIPでバックアップして、新端末でGoogleドライブなどから復元してください。

### Q: 無料で使えますか？

A: はい、完全無料です。API設定も不要です。

### Q: プライバシーは大丈夫ですか？

A: すべてのデータはあなたのブラウザ内に保存され、外部サーバーには送信されません。

## 🛠️ トラブルシューティング

### 写真が表示されない

- ブラウザのキャッシュをクリア
- ページをリロード
- ブラウザを再起動

### バックアップが失敗する

- ストレージ容量を確認
- 別のブラウザで試す
- 小分けにしてバックアップ

### カメラが起動しない

- ブラウザにカメラの許可を与える
- HTTPSで接続しているか確認
- 別のブラウザで試す

## 📝 注意事項

- ブラウザのデータ削除機能で消えてしまうので、定期的にバックアップをとりましょう
- プライベートモード/シークレットモードでは、閉じるとデータが消えます
- iOSのSafariは、ストレージ容量に制限があります

## 🎯 推奨環境

- **スマホ**: iOS 14以降（Safari）、Android 8以降（Chrome）
- **PC**: Chrome、Edge、Firefox（最新版）
- **非推奨**: Internet Explorer
